# Design

## Current Design Source

The current design source is the STOCKLAB wireframe canvas under `design/`. It includes product overview, information architecture, dashboard, personal area, stock detail, portfolio, analysis, reports, learning, admin, auth, and data architecture screens.

## Naming

The design files use STOCKLAB. The project name for implementation and documentation is Finance_lab.

## Product Shape

The design direction is a personal investing workbench:

- Dashboard first.
- Korean UI labels.
- Stock and market context close to portfolio actions.
- Thesis and review workflows as differentiators.
- Reports and learning content integrated into the investing loop.

## Implementation Guidance

Translate wireframe screens into production components before visual redesign. Keep the first pass functional and traceable to the wireframe source.
